ANGULAR CAPACITACION

ANGULAR CLI

  • Create new project: ng new appName --directory folder-name/
  • Serve project: ng serve

GIT

  • Clone repository: git clone git@github.com:jonigarciaschaff/capacitacion-angular.git
  • Check status: git status
  • Select files: git add .
  • Select files: git commit -m "first commit"
  • Push changes: git push
  • Pull changes: git pull
